# EZJ Online Landing Page

This is the landing page for EZJ Online.

## Deployment

This project is configured for automatic deployment via Vercel.

### How to push updates:

1. **Stage Changes**:
   ```bash
   git add .
   ```

2. **Commit Changes**:
   ```bash
   git commit -m "Your descriptive commit message"
   ```

3. **Push to GitHub**:
   ```bash
   git push origin main
   ```

Once pushed, Vercel will automatically detect the new commit and trigger a deployment to the live site.

> [!NOTE]
> If you encounter an `xcrun` error, please run `xcode-select --install` in your terminal to fix the macOS Command Line Tools.
